# MLOps Assignment Report Notes

The final DOCX report is generated by:

```bash
python scripts/generate_report.py
```

Expected output:

```text
reports/MLOps_Assignment_Report.docx
```

Run these commands before regenerating the report so the figures and metrics are
up to date:

```bash
python scripts/download_data.py
python scripts/run_eda.py
python -m heart_disease_mlops.train --fast
python scripts/generate_report.py
```

Add final screenshots to `reports/screenshots/` before submission.
